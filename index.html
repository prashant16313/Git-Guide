Check karein Git Install hai ya nahi

Apne terminal mein neeche diya gaya command likhein aur Enter dabayein. Isse pata chalega ki aapke system mein Git hai ya nahi.

git --version

Command chalane ke baad kya hua?

Jaise: git version 2.37.1...

Jaise: 'git' is not recognized...

Chinta na karein! Git install karna aasan hai. Neeche diye gaye instructions follow karein.

Pehli Baar Git Setup Karein

Windows Pro Tip!

Ab jab Git install ho gaya hai, to aapke paas Git Bash naam ka ek behtar terminal bhi hai. Hum salah denge ki aap ab se commands chalane ke liye iska istemaal karein.

Ise kholne ke liye Start Menu mein "Git Bash" search karein.

Bahut badhiya! Ab Git ko apna naam aur email batana zaroori hai. Yeh jaankari har commit ke saath save hoti hai, taaki pata chale ki changes kisne kiye hain. Yeh ek baar hi karna hota hai.

Neeche diye gaye commands ko terminal mein chalayein. "Your Name" aur "youremail@example.com" ko apni details se badal lein.

1. Apna Naam Set Karein

git config --global user.name "Your Name"

2. Apna Email Set Karein

git config --global user.email "youremail@example.com"
